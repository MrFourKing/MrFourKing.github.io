Навыки

Изучая инструменты и технологии, хотелось бы с их помощью поучаствовать в создании чего-то поистине грандиозного.

  • HTML / Pug / Twig

    Семантическая верстка и Pug или Twig в качестве препроцессора

  • CSS / SCSS / SASS / LESS

    Флексы, гриды, CSS-анимации. БЭМ-методология. Sass/Scss в качестве препроцессора, но знаком чуть-чуть и с Less

  • JS ES5/6

    Работа с DOM, AJAX. Классы, замыкания, промисы.

  • REACT / VUE

    Создание и рефакторинг компонентов, хуки жизненного цикла компонентов

  • GULP / webpack

    Gulp - Создание тасков с нуля, редактирование тасков под нужды текущей сборки, Webpack - базовая настройка сборки

  • Git

    Поддержка актуальности репозиториев и разрешения конфликтов (merge, commit, push, force push, pull, rebase).
